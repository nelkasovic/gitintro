# Updates holen

Deine Arbeitskopie mit neuesten Änderungen **direkt aktualisieren** kannst du mit:

```
git pull
```

Änderungen zuerst ins Repository **herunterladen** kannst du mit:

```
git fetch
```

## Zu erledigen
- [ ] Änderungen aktualisiert von Remote zu Lokal mit `git pull`
